# Seedance 2.0 快速参考卡片

## 🚀 快速开始检查清单

- [ ] 明确故事核心（一句话概括）
- [ ] 确定视频时长（4-15秒）
- [ ] 选择画面比例（9:16 / 16:9 / 2.35:1）
- [ ] 确定风格（写实/动画/科幻/水墨等）
- [ ] 准备参考素材（图片/视频/音频）
- [ ] 按时间轴拆分镜头
- [ ] 使用 @素材名 标注引用

---

## 📝 提示词公式

```
【风格】_____风格，_____秒，_____比例，_____氛围

【时间轴】
0-X秒：[镜头] + [画面] + [动作] + [特效]
X-Y秒：[镜头] + [画面] + [动作] + [特效]
...

【声音】_____配乐 + _____音效 + _____对白

【参考】@图片1 _____，@视频1 _____
```

## 🎨 氛围关键词

**光影**：逆光、侧光、剪影、轮廓光、体积光、丁达尔效应
**色调**：暖色调、冷色调、赛博朋克、复古胶片、黑白水墨
**质感**：电影级、纪录片、广告质感、MV风格、油画感

## ⚠️ 重要限制

- ❌ 不支持写实真人脸部素材
- 📊 最多12个文件（图片≤9 + 视频≤3 + 音频≤3）
- ⏱️ 视频/音频参考总时长≤15秒
- 💰 视频参考会消耗更多额度

## 🔗 多模态引用语法

```
@图片1 作为首帧

@图片3 作为角色形象参考
@视频1 参考运镜方式
@视频2 参考动作节奏
@音频1 用于配乐
```

---

## 💡 常见场景写法

### 视频延长
```
将@视频1延长X秒（生成长度选X秒）
0-X秒：[新内容]，与前视频无缝衔接
```

### 剧情颠覆
```
基于@视频1进行编辑：
【保留】原视频的_____
【修改】[具体改动]
```

### 角色替换
```
将@视频1中的人物换成@图片1
保持@视频1的运镜和动作
```

### 音乐卡点
```
参考@视频1的节奏
0-2秒：[内容]
2-4秒：音乐重拍，[内容]
```
